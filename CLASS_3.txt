

1) What is the difference between Platform dependent and platform independent?
   Platform Dependent 
   ------------------
             - Any Processor 
             - But same OS (Operating)
             - Ex: C Programming | SQL 

   Platform Independent 
   --------------------
             - Any Processor 
             - Any OS 
             - Ex: Python | Java
