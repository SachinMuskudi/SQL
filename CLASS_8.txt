


TCL OPERATIONS
---------------
- TCL MEANS (TRANSACTION CONTROL LANGUAGE)
- TCL OPERATIONS ARE USED FOR UNDO THE DATA 
- UNDO MEANS (COMMING BACK TO PREVIOUS ACTION)
- DEFAULTLY SQL WILL PERFORM THE ACTION ON THE TABLE (WHERE THE ACTION)
  WILL BE SAVED PERMANENTLY 
- TO UNDO THE DATA FROM THE TABLE WE NEED TO USE A FUNCTION CALLED 
  **ROLLBACK**
- FUNCTIONS : MEANS WHICH IS DEVELOPED BY COMMUNITY TO EXECTURE A DESIGNED TASK 
 
TCL OPERATIONS:
---------------
WE HAVE TOTAL 4 FUNCTIONS IN TCL OPERATIONS 
     - BEGIN TRANSACTION = IT IS A TECHNIQUE WHICH IS USED TO START TCL OPERATIONS 


     - ROLLBACK = THIS FUNCTION IS USED UNDO THE RECENT ACTION PERFORMED ON THE TABLE 

     - COMMIT = COMMIT IS USED TO SENT THE DATA FROM BUFFER MEMORY TO MAIN MEMORY 

     - SAVE POINT = THIS FUNCTION IS USED TO UNDO THE DATA WHICH WE TARGETED 


- USING TCL OPERATIONS ONCE WE INSERT THE DATA | IT WILL RECORDED INTO BUFFER MEMORY 
  SO WHEN WE WANT TO UNDO THE OPERATIONS (ALL DATA WILL BE UNDO TO ITS ORIGINAL POSITION)
- USING COMMIT FUNCTION IN TCL OPERATIONS WE CAN FIX THE DATA INTO MAIN MEMORY
